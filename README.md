## 구현시어려웠던점-
깃허브 연결주소를 만들 떄 교수님이 알려준 힌트만으로 조금 어려웠지만 그래도 책과 영상 시청으로 파일 실행이 아닌 외부 주소와 연결하게 하는것으로
깃허브와 연결이 되게끔 해주웠고
그 다음으로 사진 변경으로 리소스 파일을 만들어 실행 후에 리소스 파일을 불러일으킬수 있도록 코드와의 연결부분이 얼마나 중요한지 알게 되었습니다.
또한 배경 변경만으로는 조금 텍스트 배경이 너무 어울리지 않아 랜덤으로 변경한 배경색을 변수 하나로 지정헤서 나머지 텍스트배경도 바뀌게 하였습니다.
